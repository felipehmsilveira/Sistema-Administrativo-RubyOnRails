# SISTEMA ADMINISTRATIVO DE VENDAS 

Este sistema é um exemplo básico para o controle de vendas de uma empresa

Funcionalidades

* Produtos

* Clientes

* Endereços dos clientes

* Venda dos produtos

* Comissão

* Desconto em porcentagem e valor fixo

* Controle de usuário

* Perfis - Vendedor ou Gerente
